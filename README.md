# Dragon, Knight, and Wizard (Java Console Game)

Welcome to **Dragon, Knight, and Wizard**, a fantasy-themed twist on the classic Rock, Paper, Scissors game designed to play on your terminal! 

Battle against the computer, track your scores in real-time, and view performance match statistics at the end.

ICS3U Final Project
---

## 🎮 Game Rules

The game changes the core mechanics of Rock, Paper, Scissors using three fantasy archetypes:

*   🐉 **Dragon** breathes fire on the **Wizard** but is killed by the armored **Knight**.
*   🛡️ **Knight** strikes down the **Dragon** but is controlled by the mystical **Wizard**.
*   🔮 **Wizard** casts a spell on the **Knight** but is consumed by the **Dragon**.

### Summary Matrix:

| Character Choice | Beats | Loses To |
| :--- | :--- | :--- |
| **1) Dragon** | Wizard | Knight |
| **2) Knight** | Dragon | Wizard |
| **3) Wizard** | Knight | Dragon |

## 🚀 How to Run the Game

To play the game on your local machine, make sure you have the **Java Development Kit (JDK)** installed.

### 1. Clone the Repository
Open your terminal and run the following command to download the code:
```bash
git clone https://github.com/yh-cyber/DragonKnightsWizards.git
cd DragonKnightsWizards
```


### 2. Compile the Java Source Code
Compile the `Main.java` file using the Java compiler (`javac`):
```bash
javac Main.java
```

### 3. Launch the Game
Execute the compiled bytecode using the Java runtime launcher (`java`):
```bash
java Main
```
